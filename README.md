# VManager — Vestige Distributor App

A complete business management app for Vestige distributors.

## Features
- 📦 Inventory Management
- 🛒 Sales & Invoice Generator
- 💬 WhatsApp Invoice Sharing
- 📊 Reports & Analytics
- 🔔 Smart Alerts
- ⚡ AAJ KI ENERGY Videos
- 🔐 Supabase Auth + Razorpay Payments

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Configure keys in `src/VManager.jsx`
- Line ~1676: `SUPABASE_URL` and `SUPABASE_ANON`
- Line ~1696: `RAZORPAY_KEY`

### 3. Run locally
```bash
npm start
```

### 4. Build for production
```bash
npm run build
```

## Deploy to Vercel
Push to GitHub → Import on vercel.com → Deploy automatically.
